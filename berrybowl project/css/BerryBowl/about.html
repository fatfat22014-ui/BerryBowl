About the project

About BerryBowl

BerryBowl is a modern healthy food website created to help users understand nutrition, choose better meals, and build better eating habits.

Our Purpose

The purpose of BerryBowl is to make healthy eating easier, more attractive, and more accessible. Many people want to improve their lifestyle but do not know where to start. This website provides a simple starting point by offering BMI calculation, meal plans, recipes, and goal-based meal filtering.

The website is designed for everyone who wants healthy food and a better diet. It supports different goals, including weight loss, muscle gain, balanced eating, and healthy weight gain for underweight users.

Why BerryBowl?

Simple: Easy navigation and clear page structure.
Useful: BMI calculator, meal plans, recipes, and filters.
Inclusive: Supports underweight users with healthy weight gain plans.
Responsive: Works on desktop, tablet, and mobile screens.

Structure

CSS

Design

JS

Interactivity

JavaScript powers the BMI calculator, filtering, searching, and contact form validation.

DB

Data Handling

JavaScript arrays and localStorage are used to handle recipes, meals, BMI history, and messages.
