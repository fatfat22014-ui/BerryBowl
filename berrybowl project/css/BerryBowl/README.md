# BerryBowl

BerryBowl is a modern healthy food website created for an Internet Programming project.

## Features

- BMI Calculator
- Daily Meal Plans
- Healthy Recipes with Calories
- Search and filter meals by goal
- Weight Loss support
- Muscle Gain support
- Balanced Diet support
- Healthy Weight Gain support for underweight users
- About Us page
- Contact page with form validation
- Responsive design for desktop and mobile

## Technologies Used

- HTML
- CSS
- JavaScript
- JavaScript Arrays
- localStorage for basic data handling
